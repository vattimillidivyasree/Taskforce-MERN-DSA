# 📋 Taskforce – MERN DSA
<br/>
Taskforce is a basic task management app built using the MERN stack. I created this project to understand how frontend, backend, and database work together in a real application.
<br/>

## 🚀 Features
<br/>
• Add, update, and delete tasks <br/>
• Mark tasks as completed <br/>
• Backend APIs for handling data <br/>
• Simple and clean user interface <br/>

## ⚙️ Tech Stack
<br/>
Frontend: React.js <br/>
Backend: Node.js, Express.js <br/>
Database: MongoDB <br/>

## 📂 Project Structure
<br/>

```bash
Taskforce-MERN-DSA/
│── client/
│── server/
│── package.json
│── README.md
```

## ⚡ Installation & Setup
<br/>

```bash
git clone https://github.com/vattimillidivyasree/Taskforce-MERN-DSA/tree/main 
cd Taskforce-MERN-DSA
```

### Backend
```bash
cd server
npm install
npm start
```

### Frontend
```bash
cd client
npm install
npm start
```
